# SquashTM

## 環境起動
プラグインを入手してsquash-tm/docker/pluginsに配置する

**プラグインの入手先**
https://tm-en.doc.squashtest.com/v11/downloads.html


```bash
cd docker
docker compose up
```

アクセス方法

- url : http://localhost:8090/squash
- user: admin
- password: admin
